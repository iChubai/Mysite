    • 32 Registers (x0-x31), each 32-bit wide.
    • Special Registers:
    • x0 (zero): Always 0.
    • x1 (ra): Return address.
    • x2 (sp): Stack pointer.
    • x8 (s0/fp): Saved register/Frame pointer.
    • x10-x17 (a0-a7): Function arguments/return values.
    Register Name Use
    x0 zero Constant 0
    x1 ra Return address
    x2 sp Stack pointer
    x8 s0/fp Saved register/Frame pointer
    x10-x11 a0-a1 Function args/Return values
